+----------------------------------+

Discuz! 產品，無論在功能、穩定性、負載能力、安全保障等方面都居於國內外同類產品領先地位，
是全球成熟度最高、覆蓋率最大的論壇軟體系統之一。目前由合肥貳道網路科技有限公司承接維護。
